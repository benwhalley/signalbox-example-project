Adding questions

In addition to the links in the header, you can add questions by typing a new variable name into the question search box and pressing 'save and continue'... a new question will be created which you can then edit.

Ordering questions

You can add both questions and instruments to pages; ordering in the final rendered page can be controlled by using the order column of both questions and instruments attached to the page. For example, if you wanted to place an instrument between two questions, give the first question order=1, the instrument order=2 and the second question order=3.
